## 🔗 Repositorios

### 1. **zivlo** — App Móvil Flutter
- **URL:** https://github.com/ZIVLO-Team/zivlo
- **Descripción:** Punto de venta offline-first para pequeños comercios
- **Stack:** Flutter, BLoC, Hive, GoRouter
- **Estado:** ✅ En desarrollo

### 2. **zivlo-backend** — Servidor API
- **URL:** https://github.com/ZIVLO-Team/zivlo-backend
- **Descripción:** Backend para sincronización de catálogos y menú QR híbrido
- **Stack:** FastAPI, PostgreSQL, SQLAlchemy, Docker
- **Estado:** 📋 Pendiente de implementación

### 3. **zivlo-license-generator** — Generador de Licencias
- **URL:** https://github.com/ZIVLO-Team/zivlo-license-generator
- **Descripción:** Generador de licencias JWT con firma HMAC para validación offline
- **Stack:** Python, JWT, HMAC-SHA256
- **Estado:** 📋 Pendiente de implementación
